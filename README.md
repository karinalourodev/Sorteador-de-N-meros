# 🎲 Sorteador de Números

Um simples sorteador de números aleatórios feito em **HTML, CSS e JavaScript**, com animação de dado girando para deixar mais divertido.

---

## 🚀 Funcionalidades
- Sorteia um número entre dois valores definidos pelo usuário.
- Exibe o resultado diretamente na página.
- Animação do dado girando ao clicar em **Sortear**.
- Interface estilizada com gradiente e efeitos modernos.
